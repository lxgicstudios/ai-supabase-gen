Generate Supabase RLS policies from a Prisma schema file

## Install

```bash
npm install -g ai-supabase-gen
```

## Usage

```bash
npx ai-supabase-gen ./path/to/file
```

## Setup

```bash
export OPENAI_API_KEY=sk-...
```
